# P-fitness

## 概要
- フィットネスをタスクとして追加し、完了するとアプリ内ポイントと経験値を獲得
- ご褒美を設定
- ポイントを消費してご褒美を獲得できる
- 経験値に応じてレベルアップ

## 実装予定の機能
- ログイン・ログアウト機能
- フィットネス登録・削除・更新機能
- ご褒美登録・削除・更新機能
- フィットネス完了やご褒美獲得に伴うポイントの増減処理

## 画面の種類
- ログインページ
- ログアウトページ
- アカウントページ(登録情報の確認)
- ホームページ(フィットネスの登録や削除・更新・完了ができる)
- リワードページ(褒美の登録・更新・削除、獲得ができる)
